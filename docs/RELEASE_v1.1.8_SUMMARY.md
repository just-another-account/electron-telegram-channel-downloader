## 🎯 主要功能

### 1. 多媒体消息支持 📱
- **问题**: 之前只能下载多媒体消息中的第一个文件
- **解决**: 现在支持下载消息中的所有媒体文件（媒体组/相册）
- **特性**:
  - 自动检测媒体组（groupedId）
  - 下载组内所有媒体文件
  - 智能文件命名（序号后缀）
  - 去重处理避免重复下载

### 2. 对话筛选器系统 🔍
- **位置**: 左侧对话列表上方
- **功能**:
  - 实时搜索对话名称、标题、ID
  - 按类型筛选（频道、群组、用户）
  - 筛选结果统计显示
  - 智能空状态处理
  - 一键重置功能
